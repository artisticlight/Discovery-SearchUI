Using The "More Like This" Feature

When you perform a search and the results are returned, you can browse the results and find a scene that covers a small area of interest to you.

Once you have done so, you can then click the "More Like This" button in the Scene Detail, and Vertex will perform another search using the path and frame of the selected scene. The results will now contain only scenes with the same path and frame.
