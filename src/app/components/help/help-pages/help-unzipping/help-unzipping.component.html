Using Zipped Product Files

Some dataset results contain bundled and compressed, or zipped, files. Vertex allows for the contents of these zipped files to be examined and individual files within the zip archive to be added to the cart or download, in addition to permitting the same with the entire zip file itself.

Zipped files will be denoted by a right-pointing caret to the left of the filename. Clicking the caret will expand the list to show all of the individual components within the zip archive. You can then manipulate each of the components as though they were independent files.
